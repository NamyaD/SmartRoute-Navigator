## 🌟 Overview

**SmartRoute Navigator** is a C++-based navigation system inspired by modern mapping applications. It models cities and roads as a weighted graph and applies algorithms such as **Dijkstra's Algorithm, BFS, and DFS** to compute optimal routes and analyze connectivity.

---


## 💡 Why I Built This

I built SmartRoute Navigator to bridge the gap between theoretical Data Structures & Algorithms and real-world software applications. While studying graph algorithms such as Dijkstra's Algorithm, BFS, and DFS, I wanted to apply these concepts to a practical problem that is widely used in everyday life — navigation and route optimization.

This project helped me strengthen my understanding of graph-based problem solving, object-oriented design, STL containers, priority queues, and file handling while building a modular software system.

---

## ⚡ Features

| Feature | DSA / Concept Used |
|---|---|
| Add cities | Hash Map |
| Add roads | Adjacency List |
| Shortest route | Dijkstra's Algorithm |
| Cheapest route | Dijkstra's Algorithm |
| Fastest route with traffic | Weighted Graph |
| Reachable cities | BFS |
| Explore connected map | DFS |
| Save and load map | File Handling |
| Route summary | Path Reconstruction |

## 🚀 Installation

### Prerequisites
- C++ Compiler (g++)
- VS Code / CodeBlocks / CLion

### Clone Repository

```bash
git clone https://github.com/NamyaD/SmartRoute-Navigator.git
```

### Compile

```bash
g++ main.cpp Graph.cpp SmartNavigator.cpp -o SmartRouteNavigator
```

### Run

```bash
./SmartRouteNavigator
```
